# Configuration Files

This directory contains the central environment variables and configuration settings for the evaluation pipelines.

* **`pass_k_eval.env`**: Settings and model parameters for the standard Pass@K evaluation pipeline.
* **`cot_judge.env`**: Configurations for the LLM-as-a-Judge pipeline (judge models, temperatures, etc.).
* **`extractor.env`**: Environment variables required for filtering and extracting correct generations.
